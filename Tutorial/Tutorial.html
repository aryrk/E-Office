Home

Bagian ini adalah interface pertama saat membuka website officia, user dapat mengakses menu login dari tombol "login" pada bagian pojok kanan atas website atau tombol "how to use" untuk membaca panduan menggunakan officia.
Pada bagian bawah, terdapat tombol kiritk dan saran yang berguna apabila terdapat user yang memiliki masalah ataupun kebingungan tentang officia.

Fitur
  • Animasi interaktif yang akan muncul saat suatu class muncul pada layar
  • Icon interaktif saat mouse hover
  • Fitur contact yang memperbolehkan user untuk mengirimnkan email pada developer

Login

Pada tampilan ini, user dapat memasukan username dan password mereka untuk mengakses officia.
(Akun hanya dapat didaftarkan oleh developer atau adminisrator dari setiap perusahaan.)

Fitur
  • Registrasi
  • Fitur hide password

Tampilan Utama

Setelah login, user akan dihadapkan pada tampilan utama yang berisi data dasar mereka, hal ini bertujuan untuk memastikan bahwa mereka masuk pada akun yang benar.
Tampilan ini terhubung dengan menu absensi dan pengumuman.

Fitur
  • Overview, jika ditekan maka website akan menunjukan status user yang dikalkulasikan dari ketertiban absen dan kinerja kerja.
  • Tugas mingguan, bila ditekan maka website akan menunjukan tugas tugas yang ditentukan perusahaan perhari/perminggu
  • Mode gelap yang akan mengubah tampilan website menjadi hitam (telah dilaporkan bahwa fitur ini tidak bekerja pada beberapa perangkat)

Absensi

Menu ini biasa digunakan untuk melakukan absen datang dan pulang yang akan mempengaruhi status ranking pada tampilan adminisrator.
Pengguna dapat meminta cuti langsung dari aplikasi dan akan terkirimkan pada adminisrator, pengguna akan mendapat pemberitahuan apabila diberikan izin/tidaknya bercuti.

Fitur
  • Tombol shortcut interaktif
  • Jam digital yang menunujukan waktu akurat walau jam pada komputer tidak benar
  • Menu absen yang dapat digunakan untuk absen datang dan pulang
  • Cuti online yang memudahkan pegawai meminta cuti langsung pada aplikasi
  • Databse absen yang akan menampilkan seberapa sering anda melakukan absen, dan kapan anda tidak hadir

Pengumuman

Di menu ini, pengguna dapat melihat segala pengumuman yang dikimkan oleh perusahaan masing masing.
Dengan begitu, semua karyawan dapat mengetahui segala pemberitahuan dengan cepat, dimanapun kapanpun.

Fitur
  • Icon officia yang bergerak

Adminisrator

Tampilan ini merupakan UI admin perusahaan, pada tampilan awal user dapat melihat rank para pegawainya.

Fitur
  • Setting jam absen

    • Fitur ini dapat merubah jam absen minimal dan maksimal untuk mengatur absen karyawan. Apabila salah satu karyawan melakukan absensi diluar jam yang telah ditentukan, maka sistem akan menganggap karyawan tersebut telat absen

  • Tambah pengumuman

    • Disini admin dapat mengirimkan pengumuman pengumuman pada karyawannya dengan cara mengisi form dan tujuan pengumuman tersebut.

  • Tambah tugas

    • Bagian ini memungkinkan admin untuk mengirimkan tugas pada karyawan dan memiliki sistem yang sama dengan kirim pengumuman
