Bagian ini adalah interface pertama saat membuka website officia, user dapat mengakses menu login dari tombol "login" pada bagian pojok kanan atas website atau tombol "how to use" untuk membaca panduan menggunakan officia.

Login

Pada tampilan ini, user dapat memasukan NIK dan password mereka untuk mengakses officia.
(Akun karyawan hanya dapat didaftarkan oleh developer atau adminisrator dari setiap perusahaan.)

Fitur
  • Registrasi Akun Perusahaan

    • Form ini akan membuat akun administrasi yang nantinya dapat digunakan untuk mendaftarkan akun karyawan

  • Fitur lupa sandi

    • Seorang karyawan dapat mereset sandi mereka dengan cara memasukan data data diri yang sebelumnya telah didaftarkan admin

User Interface Admin

Setelah melakukan login admin, user akan dihadapkan pada tampilan rank absen yang akan muncul jika sistem memiliki cukup data karyawan pada akun tersebut.

Fitur
  • Setting Jam Absen.

    • Tampilan ini memungkinkan admin untuk mengatur jam minimal dan maksimal absen harian, yang nantinya akan mempengaruhi aturan sistem absen perusahaan tersebut.

  • Izin Cuti

    • Jika ada karyawan yang mengirimkan permintaan cuti, admin dapat menerima/menolak permohonan tersebut.

  • Log Cuti Karyawan

    • Bagian ini menampilkan semua permintaan cuti beserta status aktifnya.

  • Pengumuman

    • Bagian ini memperbolehkan admin untuk mengirimkan pengumuman baik pada seluruh karyawannya atau pada divisi divisi tertentu.

  • Registrasi Karyawan

    • Form ini digunakan untuk mendaftarkan karyawan pada akun admin anda.

  • List Karyawan

    • Disini user dapat melihat semua karyawan yang telah didaftarkan pada akun tersebut, pengguna juga dapat melakukan penghapusan akun.

  • Log Absen Karyawan

    • User dapat melihat data asben karyawan karyawannya melalui tampilan ini.

  • Upload Tugas

    • Pada tampilan ini user dapat mengupload tugas untuk karyawan karyawannya, baik kepada seluruh karyawan ataupun salah satu divisi.

  • Log Pengiriman Tugas

    • Data data tugas yang telah dikirim akan disimpan pada tampilan ini, user juga dapat menghapus salah satu tugas tersebut.

Tampilan Utama

Setelah login, user akan dihadapkan pada tampilan utama yang berisi data diri mereka, hal ini bertujuan untuk memastikan bahwa mereka masuk pada akun yang benar. Pengguna juga dapat mengubah foto profil mereka dengan cara menekan tombol ubah.
Tampilan ini terhubung dengan menu absensi dan pengumuman.

Fitur
  • Overview.

    • Tampilan ini akan muncul bila user sudah melakukan absen harian.

  • Tugas

    • Disini user dapat melihat daftar tugas yang diupload admin perusahaan.

Absensi

Bagian ini merupakan tampilan yang menghubungkan dengan absensi, permintaan cuti, dan data absen mandiri.

Fitur
  • Absensi

    • Disini user dapat melakukan absen harian sesuai dengan jam yang telah ditentukan admin.

  • Cuti

    • Pengguna dapat mengirimkan permintaan cuti melalui tampilan ini, mereka juga dapat menghapus permohonan apabila belum dikonfirmasi admin.

  • Log Absen Pribadi

    • Disini user dapat melihat log pengiriman absen dengan detail.

Pengumuman

Di menu ini, pengguna dapat melihat segala pengumuman yang dikimkan oleh perusahaan masing masing.
Dengan begitu, semua karyawan dapat mengetahui segala pemberitahuan dengan cepat, dimanapun kapanpun.
